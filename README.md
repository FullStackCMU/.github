# 🚀 CollabReflect

> **Team Evaluation Platform** | Web Application for Student & Instructor Collaboration
> Developed by **FullStackCMU** Team

---

## 📌 About The Project
**CollabReflect** คือระบบประเมินการทำงานกลุ่มและสะท้อนผลการเรียนรู้ (Peer & Team Evaluation Platform) ที่ออกแบบมาเพื่อช่วยให้นักศึกษาและอาจารย์สามารถจัดการ ติดตาม และประเมินผลการทำงานร่วมกันได้อย่างมีประสิทธิภาพ

---

## 🛠️ Tech Stack & Architecture

### **Frontend**
* **Framework:** React / Vite (TypeScript)
* **Styling:** Tailwind CSS

### **Backend**
* **Runtime/Framework:** Node.js (TypeScript) / Express / Hono
* **Database & ORM:** PostgreSQL + Drizzle ORM

### **DevOps & Infrastructure**
* **Containerization:** Docker & Docker Compose
* **Version Control:** GitHub Organizations
* **Deployment:** Ubuntu VM / Docker Hub
